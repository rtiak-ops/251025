# ToDo アプリ

## 📌 概要
このアプリは、タスクの追加・編集・削除ができるシンプルな ToDo 管理アプリです。  
React（TypeScript）＋ FastAPI ＋ PostgreSQL ＋ Docker で構築しています。

---

## ✨ 主な機能
- タスクの追加
- タスクの編集
- タスクの削除
- 完了状態の切り替え
- 永続化（PostgreSQL）

---

## 🛠 使用技術（Tech Stack）
- **Frontend:** React / TypeScript / Tailwind CSS  
- **Backend:** FastAPI  
- **Database:** PostgreSQL  
- **Infrastructure:** Docker / docker-compose
